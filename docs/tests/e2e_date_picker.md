# E2E: Date Picker（最小）

目的: inline初期化撤去後も、外部JS（date_picker_init.js）で従来どおり動作するか確認。

前提: 日付入力フィールド（`.nr-date-picker[data-wrap]` または `.js-date`）が存在する画面。

## シナリオ
1. ページを開く → コンソールエラーが無い
2. 日付入力をクリック → カレンダーが開く
3. 適当な日付を選択 → 表示が `YYYY-MM-DD` に変わる（altInputに日本語表示があれば従来どおり）
4. フォーカス遷移（Tab）で不自然な挙動がない
5. 送信（GET/POSTどちらでも可）→ 200

## 耐性確認
- ベンダー遅延時（flatpickrを意図的に遅延）でも例外が出ず、入力が通常のテキストとして機能（no-op）
- 再実行（DevToolsから初期化処理を再実行）しても例外が出ない（重複初期化回避）
